## How to Use｜使用方法

1. Click **Code → Download ZIP**.
2. Extract the downloaded file.
3. Open the project folder in Godot.

点击右上方的 **Code → Download ZIP** 下载仓库，解压后使用 Godot 打开需要的示例工程。

不同工程可能使用不同版本的 Godot。  
Different projects may require different Godot versions.
