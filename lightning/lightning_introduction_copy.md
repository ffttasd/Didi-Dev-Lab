# 闪电效果项目介绍文案

今天我们来聊聊
如何在 Godot 里面做一个闪电效果

先来看一下现在的效果

这是一道会自己变化的闪电
我们把速度放慢一点
看一下它到底是怎么动的

可以看到
它并不是一张固定的图片
中间这条线一直在变化
旁边还会随机长出一些分叉

首先创建一个 QuadMesh
把它当成闪电的载体

然后给它套上一个 Shader

我们先做中间的主干
根据当前像素的纵坐标
算出闪电中心线的位置

现在它是一条直线
看起来有点太规整了

直接加一点噪波
让它左右晃动起来

再叠加几层不同频率的噪波
这个形状就会随机很多

这里再加一点正弦变化
让它不要完全停下来

现在主干的基本形状就有了

接下来做一下发光

我们计算当前像素离中心线有多远
用一个比较窄的范围
切出中间的白色核心

然后再扩大一点范围
做一层蓝色的光晕

这样就能得到一个这样的效果

现在来加分叉

我在闪电上面随便取几个位置
让它从这些地方往两边长出去

分叉的方向可以直接用一个值控制
负数往左
正数往右

再用同样的方法给分叉加一点噪波
它就不会像树枝一样笔直了

我们给分叉做一个渐变
让它从根部比较粗
到末端慢慢消失

目前看起来还可以
但是它一直亮着
没有什么闪电的感觉

下面加一个闪烁

先把时间乘上一个频率
每一个周期生成一个随机数
用这个随机数决定这一轮要不要闪

现在它已经会随机亮一下了
不过这个亮度变化有点生硬

我们再加几个延迟的脉冲
模拟闪电第一次亮起之后
又回闪几下的效果

然后叠加一点快速的噪声
让它内部也跟着抖动

这样就有点那个意思了

现在调整一下核心和光晕的能量
让它亮起来

如果只需要一个屏幕上的特效
到这里其实就已经可以用了

但为了让它更有感觉
我们再加一盏 OmniLight3D

在代码中获取刚才的闪烁强度
然后把它同步给灯光

闪电亮起来的时候
地面也会跟着被照亮

我们把灯光的闪烁速度放慢一点
可以看到它和闪电基本是同步的

最后简单整理一下参数

闪电的颜色、粗细、扭曲程度
还有分叉的数量和长度
都可以直接在面板里面调整

换一个颜色看一下

紫色也可以
红色也可以
只要修改核心色和光晕色就行了

至此我们就完成了
一个可以实时变化的闪电效果

它不需要序列帧贴图
也不需要提前画好每一帧动画
直接通过 Shader 就可以生成
